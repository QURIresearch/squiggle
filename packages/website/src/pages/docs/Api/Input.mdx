# Input

Inputs are now only used for describing forms for [calculators](./Calculator.mdx).

### Input.text

```
Input.text: ({
  name: string,
  description?: string,
  default?: string,
}) => input
```

### Input.textArea

```
Input.textArea: ({
  name: string,
  description?: string,
  default?: string,
}) => input
```

### Input.checkbox

```
Input.checkbox: ({
  name: string,
  description?: string,
  default?: boolean,
}) => input
```

### Input.select

```
Input.select: ({
  name: string,
  description?: string,
  default?: string,
  options: string[]
}) => input
```
